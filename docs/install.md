# Install

1. Locate your mpv config directory:
   - **Windows:** `%APPDATA%\mpv\`
   - **macOS:** `~/.config/mpv/`
   - **Linux:** `~/.config/mpv/`
2. Copy the **contents** of this repo's `portable_config/` into that directory.
3. Restart mpv.

## Updating
Re-copy `portable_config/` over your existing files.

## Uninstall
Delete `scripts/uosc/`, `fonts/Geist-*.ttf`, and `script-opts/uosc.conf` from your mpv config directory.
