# Customize

Edit `script-opts/uosc.conf` and restart mpv.

## Knobs
| Option | Values | Default | What it does |
|---|---|---|---|
| `liquid_glass_theme` | `dark`, `light` | `dark` | Glass tint and content color |
| `liquid_glass_intensity` | `0.5`–`1.5` | `1.0` | Multiplier on all alpha values |
| `liquid_glass_accent` | hex color | `E8553A` | Accent color (progress fill, selection) |
| `liquid_glass_show_frost_noise` | `yes`, `no` | `yes` | Toggle the noise texture layer |

## Power users
Edit `scripts/uosc/lib/theme.lua` directly to change individual token values.
