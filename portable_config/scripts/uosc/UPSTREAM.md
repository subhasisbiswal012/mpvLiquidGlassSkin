# Upstream pin

This directory is a fork of [tomasklaen/uosc](https://github.com/tomasklaen/uosc).

**Pinned to:** 5.9.2

When rebasing onto a newer upstream:
1. `git clone --depth 1 --branch <new-tag> https://github.com/tomasklaen/uosc.git ../uosc-upstream`
2. Three-way merge our changes against the new upstream — see `docs/contributing.md`.
3. Update this file with the new pin.
